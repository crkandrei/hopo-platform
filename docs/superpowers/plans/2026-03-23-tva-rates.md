# TVA Rates Nomenclator & Product VAT Support — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a global TVA rates nomenclator (Super Admin managed) and attach a configurable VAT class to products so fiscal receipts sent to HopoFiscalBridge carry the correct `vatClass` per item instead of the hardcoded `1`.

**Architecture:** Two repos change: `hopo-platform` (Laravel 12) adds the `tva_rates` table, product relation, CRUD UI, and updates `prepareFiscalPrint` endpoints to include `vatClass` per product item; `HopoFiscalBridge` (TypeScript/Node.js) accepts the optional `vatClass` field and uses it when generating the Datecs `.txt` file. Packages and play-time always use `vatClass: 1`.

**Tech Stack:** PHP 8.2 / Laravel 12 / Blade / MySQL; TypeScript / Node.js / Express / Jest / Zod

**Spec:** `docs/superpowers/specs/2026-03-23-tva-rates-design.md`

```php
<?php

namespace App\Http\Controllers;

use App\Models\TvaRate;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Auth;

class TvaRateController extends Controller
{
    private function requireSuperAdmin(): void
    {
        $user = Auth::user();
        if (!$user || !$user->isSuperAdmin()) {
            abort(403, 'Acces permis doar pentru Super Admin');
        }
    }

    public function index()
    {
        $this->requireSuperAdmin();
        $tvaRates = TvaRate::orderBy('vat_class')->get();
        return view('tva-rates.index', compact('tvaRates'));
    }

    public function create()
    {
        $this->requireSuperAdmin();
        return view('tva-rates.create');
    }

    public function store(Request $request)
    {
        $this->requireSuperAdmin();

        $validated = $request->validate([
            'name'       => ['required', 'string', 'max:255'],
            'percentage' => ['required', 'numeric', 'min:0', 'max:100'],
            'vat_class'  => ['required', 'integer', 'min:1', 'max:9', 'unique:tva_rates,vat_class'],
            'is_active'  => ['sometimes', 'boolean'],
        ]);

        TvaRate::create([
            'name'       => $validated['name'],
            'percentage' => $validated['percentage'],
            'vat_class'  => $validated['vat_class'],
            'is_active'  => isset($validated['is_active']) ? (bool) $validated['is_active'] : true,
        ]);

        return redirect()->route('tva-rates.index')->with('success', 'Cota TVA a fost adăugată.');
    }

    public function edit(TvaRate $tvaRate)
    {
        $this->requireSuperAdmin();
        return view('tva-rates.edit', compact('tvaRate'));
    }

    public function update(Request $request, TvaRate $tvaRate)
    {
        $this->requireSuperAdmin();

        $validated = $request->validate([
            'name'       => ['required', 'string', 'max:255'],
            'percentage' => ['required', 'numeric', 'min:0', 'max:100'],
            'vat_class'  => ['required', 'integer', 'min:1', 'max:9', 'unique:tva_rates,vat_class,' . $tvaRate->id],
            'is_active'  => ['sometimes', 'boolean'],
        ]);

        $tvaRate->update([
            'name'       => $validated['name'],
            'percentage' => $validated['percentage'],
            'vat_class'  => $validated['vat_class'],
            'is_active'  => isset($validated['is_active']) ? (bool) $validated['is_active'] : false,
        ]);

        return redirect()->route('tva-rates.index')->with('success', 'Cota TVA a fost actualizată.');
    }

    public function destroy(TvaRate $tvaRate)
    {
        $this->requireSuperAdmin();

        if ($tvaRate->products()->exists()) {
            return redirect()->route('tva-rates.index')
                ->with('error', 'Nu se poate șterge o cotă TVA care este folosită de produse.');
        }

        $tvaRate->delete();

        return redirect()->route('tva-rates.index')->with('success', 'Cota TVA a fost ștearsă.');
    }
}
```
